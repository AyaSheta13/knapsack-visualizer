# Knapsack Algorithm Visualizer

An interactive web application that demonstrates the 0-1 Knapsack problem solution using dynamic programming.

## Features

- Interactive item management (add/remove items)
- Step-by-step algorithm visualization
- Dynamic programming table display
- Real-time solution calculation
- Responsive design

## How to Use

1. Set the knapsack capacity (1-10)
2. Add items with values (0-50) and weights (1-10)
3. Click "Start Algorithm" to see the step-by-step solution
4. View the optimal items selection

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Algorithm

This implements the 0-1 Knapsack solution using dynamic programming with O(n*W) time complexity.
